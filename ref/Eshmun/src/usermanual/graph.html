Graph Manipulating: Single Kripke Structure

By default, Eshmun loads in Single Kripke Structure mode.

You can create a new Kripke Structure, by going to the File menu -> New -> Kripke Structure (Ctrl + k).


States: You can add a new states by any one of these ways:

  • Insert Menu -> State
  • Right Click on a Free Spot -> New State
  • Press Insert button with focus inside the Graph Panel.

A dialog will appear, you need to provide a unique name for the state, as well as the propositional labels it satisfies (comma separated).


Transitions: You can add a new transition by any one of these ways:

  • With a State selected, Go to Insert Menu -> Transition
  • Right Click on a State -> Add Transition
  • With a State selected, Press Shift + insert.

The state selected before attempting to add a transition would be the start state, after attempting the add the next state selected would be the end state of the new transition.


Editing & Removing

To Edit and remove a state or a transition, select it, then either go to the Edit menu, or right click on it.

Also, You can edit a state by simply double clicking on it, or selecting it then pressing F2. You could also delete a state or a transition by selecting it then pressing delete.


Structure Name

You are advised to give your structures meaningful names, this will help later in case you decided to use the structure in a Concurrent Kripke Structure. If the structure contains different processes it is advised to state these processes in the structure name (comma separated inside parenthesis). For example: mutex(A, B, C).


To change the structure name, go to the Edit Menu -> Edit Structure Name or press F4. You will be prompted to enter the structure name. Note that for Concurrent Kripke Structure, the procedure for changing the name and the constraints on the names are different.


Shared Variables

You can defined shared variables (that take values) inside the structure, the shared variable is part of the state and is written among its propositional labels.

The syntax for shared variables is <variable name>{<process index>,<process index>,...}:=<value> If process indices are omitted, then the shared variable is considered shared between all the process in the current structure.


Formatting and Spacing

You can drag and drop states to change your positions, simply hold the left mouse button while on a state to start moving it.

You might want to auto space and format the graph, to do so, go to View menu -> Auto format. If the resulting Graph is not pleasing enough, you could alter few states manually, then attempt to auto format it again.

Please notice the larger the structure the greater the time needed to format it.


Clear, Undo, & Redo

You can clear the current structure by pressing right click on any free point, then choosing clear (Ctrl + l).

You can undo/redo any changes to the current structure by going to the Edit menu -> Undo / Redo (Ctrl + z, Ctrl + y respectively).

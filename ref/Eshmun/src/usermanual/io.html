Saving and Loading

You can save and then load any model, the save perserves information and format / spacing.


To Save the whole structure, go to File Menu -> Save (Ctrl + s), a dialog will appear asking for the output file.


You could also save the current structure in a Concurrent-Kripke Structure, Open the tab for that specific structure, right click on any free point, and choose save. A similar dialog will appear.


To open a structure from a file, go to File Menu -> open (Ctrl + o), a dialog will appear asking for the input file.


You could also load a single structure into a Concurrent Kripke Structure, simply choose the designated tab, right click on any free point, and choose load. This will replace the contents of the current tab.

Model Check

You can model check any model after inputing its specifications, to do so simply go to the Tools Menu -> Model Check (F8). Model Check will be carried out for the entire open structure.


After model checking is complete, the result is displayed in the status panel (bottom right corner of the screen), with the time required in nano seconds.


Model Checking a concurrent Kripke structure will check all the pairs against their specifications.


Global Model Check

Eshmun supports a variant of Model Checking as well. Global Model Checking allows to model check each state against a provided CTL formula (could be different than the structure specifications). The states that satisfy the formula will be indicated visually.


To use Global Model Check go to the Tools Menu -> Global Model Check. A dialog will popup asking you for the CTL formula to be used, you can type it into the box, the box has syntax coloring and error detection help. When you are satisfied with the formula, click OK to proceed.


The states that satisfy the given formula will be shaded with stripes of grey, states not shaded do not satisfy the formula.


Global Model Check can be used in order to identify certain states that may cause repair to fail. It can be used to help modify the structure accordingly.
